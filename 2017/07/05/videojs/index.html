videojs基本事件

  • durationchange：视频时长改变了（只有视频加载后，时长会由“NaN”变成音频/视频的实际时长）
  • ended：视频播放结束（currentTime == duration）
  • error:视频加载期间出错
  • fullscreenchange：进入/退出全屏状态
  • loadeddata：当浏览器已加载视频的当前帧时
  • loadedmetadata：当浏览器已加载视频的元数据时
  • loadstart：当浏览器开始查找视频时，开始加载
  • pause：视频暂停
  • play：视频播放
  • progress：下载视频
  • timeupdate：播放位置更改时
  • volumechange：音量已经更改
